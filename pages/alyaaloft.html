Project Overview

AlyaAloft is a sophisticated PDF document analysis and question-answering application that leverages advanced AI models to provide detailed responses to user queries about PDF documents. The system is designed to extract, process, and analyze PDF content in depth, enabling users to ask natural language questions and receive comprehensive, context-aware answers.

Key Features

  • Document Processing:
    • Text extraction from PDFs with OCR support for scanned documents
    • Document structure analysis to identify sections and hierarchies
    • Semantic chunking and embedding for efficient retrieval
    • Vector-based storage for fast similarity search
  • AI Response Generation:
    • Specialized handling for different query types (summary, detailed questions)
    • Smart fallback mechanisms for unclear or ambiguous queries
    • Comprehensive responses with evidence from the document
    • Multiple AI models with fallback options for robustness
  • User Experience:
    • Intuitive web interface for document uploads and chat
    • Real-time WebSocket communication for interactive responses
    • Chat history storage with context preservation
    • Detailed document metadata and organization

Technical Implementation

AlyaAloft is built with a layered architecture that separates concerns and provides clear responsibility boundaries:

  • Frontend: Streamlit web UI and REST API clients for user interaction
  • API Layer: FastAPI endpoints for document upload, queries, and WebSocket
  • Service Layer: Document processing, AI response generation, and structure analysis
  • Storage Layer: JSON-based storage for documents, vector embeddings, and chat history

AI Components

The system integrates several advanced AI models for different aspects of document processing:

  • Llama-2-7b-chat: Primary LLM for detailed reasoning and question answering
  • Flan-T5-Base: Used for summarization and as a fallback option
  • Sentence Transformers: For generating text embeddings for semantic search
  • Specialized prompt templates for different query types and document analysis

Results & Innovations

AlyaAloft represents significant innovations in document processing and AI-driven information retrieval:

  • Advanced query recognition for understanding user intent
  • Sophisticated prompt engineering for detailed and accurate responses
  • Robust fallback mechanisms to handle edge cases and errors
  • Document structure-aware processing for context enhancement
  • End-to-end solution from document upload to interactive questioning
